                    BORE TUNNEL - SIMPLE GUIDE
════════════════════════════════════════════════════════════════

FOR END USERS (Your customers):
────────────────────────────────────────────────────────────────

WINDOWS:
  1. Download: bore-tunnel-setup.exe
  2. Double-click it
  3. Click "Next" → "Install" → "Finish"
  4. Find "Bore Tunnel" in Start Menu
  5. Launch and login!

macOS:
  1. Download: bore-tunnel.dmg
  2. Double-click it
  3. Drag app to Applications folder
  4. Open Applications → Double-click Bore Tunnel
  5. Login and use!

LINUX (Easy - all distros):
  1. Download: bore-tunnel.AppImage
  2. Right-click → Properties → Permissions
  3. Check "Allow executing as program"
  4. Double-click to run
  5. Login and use!

LINUX (Ubuntu/Debian):
  1. Download: bore-tunnel.deb
  2. Double-click it
  3. Click "Install" → Enter password
  4. Find in applications menu
  5. Login and use!

LINUX (Fedora/RHEL):
  1. Download: bore-tunnel.rpm
  2. Double-click it
  3. Click "Install" → Enter password
  4. Find in applications menu
  5. Login and use!
